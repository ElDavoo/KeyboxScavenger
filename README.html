KeyboxScavenger

Systemless module with two independent daemons:

  1. Keybox updater (kbsd): HEAD-first check every 3 hours (immediate first run), conditional download, and atomic write to /data/adb/tricky_store/keybox.xml preserving attributes/context.
  2. PlayIntegrityFix watcher (pifd): parses SECURITY_PATCH from /data/adb/modules/playintegrityfix/custom.pif.prop and runs /system/bin/sh /data/adb/modules/playintegrityfix/action.sh when stale (immediate then daily until value changes).

Crash isolation is guaranteed by separate process/lock/state/log/trap handling.
