# This module runs two independent daemons.
#
# 1) keybox daemon (kbsd):
#    - Immediate check at startup, then every 3 hours.
#    - HEAD first on keybox URL, GET only when modified.
#
# 2) pif daemon (pifd):
#    - Reads SECURITY_PATCH from:
#      /data/adb/modules/playintegrityfix/custom.pif.prop
#    - Runs /data/adb/modules/playintegrityfix/action.sh with /system/bin/sh
#      when SECURITY_PATCH is at least 30 days old.
#    - Runs immediately on stale detection and then once per day until patch changes.
#
# Optional runtime overrides (environment variables):
#   KEYBOX_URL=https://www.davidepalma.it/pib/keybox.xml
#   KEYBOX_TARGET=/data/adb/tricky_store/keybox.xml
#   KEYBOX_INTERVAL_SECONDS=10800
